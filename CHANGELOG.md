# Changelog

## 0.1.0 (2026-03-22)

- Initial release
- `Debounce.Create` for creating debounced actions with configurable delay
- `DebouncedAction` with leading and trailing edge support
- `Throttle.Create` for creating throttled actions
- `ThrottledAction` with thread-safe execution limiting
- Async support with `Func<Task>` overloads
- Manual `Cancel` and `Flush` control
